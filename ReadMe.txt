All the source code is included in this zip file along with the images.

To run this application.
1) Install Microsoft SQL Server Developer Edition
2) Install SSMS
3) Restore the database from the backup file named "S3G2-PVFDB.bak"
4) Extract the zip file contents into folder
5) Install and open Visual Studio Community Edition
6) Load the project solution file from the extracted folder.
7) Set up the model entities connection in the server explorer.
8) Build the solution 
9) Run the solution
